Google Drive Image Gallery

View and manage images from your Google Drive folder

You need to authenticate with Google to access your Drive images

Setup Instructions

  1. Create a new Google Apps Script project
  2. Replace the default code with the provided Google Apps Script code
  3. Deploy the script as a web app with permissions set to "Anyone, even anonymous"
  4. Replace the SCRIPT_URL in the JavaScript code with your deployment URL
  5. Create a folder in your Google Drive named "Website Images" and add some images to it
  6. Click "Sign in with Google" and authorize the application
